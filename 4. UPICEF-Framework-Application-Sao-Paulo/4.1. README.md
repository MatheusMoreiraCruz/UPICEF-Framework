# Application of the UPICEF Framework – City of São Paulo

This folder presents the empirical application of the UPICEF Framework to the City of São Paulo.
It documents the results obtained in each activity of the framework application process,
ensuring transparency, traceability, and reproducibility.

---

## Objective

To demonstrate how UPICEF can be applied in a real-world context and to make the results
openly available for inspection, reuse, and comparison.

---

## Application Scope

- City: São Paulo (Brazil)
- Planning Instruments:
  - Strategic Planning (Agenda 2030)
  - Goals Plan (Plano de Metas)
  - Multi-Year Plan (PPA)
  - Budget Guidelines Law (LDO)
  - Annual Budget Law (LOA)

---

## Structure

Each subfolder corresponds to one activity of the UPICEF application process:

1. Document Collection  
2. Component Extraction  
3. Completeness Assessment  
4. Qualitative and Quantitative Analysis  
5. Discussion and Interpretation
