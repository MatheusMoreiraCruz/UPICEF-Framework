# Activities, Inputs, and Outputs

## Activity 1 – Document Collection

### Description
Collection of officially established urban planning instruments from institutional sources,
such as government portals and transparency websites.

### Inputs
- Institutional websites
- Official planning repositories
- Publicly available documents

### Outputs
- Consolidated set of planning instruments

---

## Activity 2 – Extraction of Structural Components

### Description
Identification and mapping of structural components present in each planning document,
focusing exclusively on formal structure.

### Inputs
- Planning instruments
- Structural component identification criteria

### Outputs
- Inventory of components identified in each document

---

## Activity 3 – Comparison with Minimum Components

### Description
Systematic comparison between the components identified in the documents and the minimum
components defined by the UPICEF framework.

### Inputs
- Component inventory
- Minimum components reference model

### Outputs
- Completeness assessment matrices
- Identification of structural gaps

---

## Activity 4 – Qualitative and Quantitative Analysis

### Description
Analysis of comparison results to identify patterns, strengths, and weaknesses across
planning instruments.

### Inputs
- Comparison results
- Completeness matrices

### Outputs
- Completeness indicators
- Aggregated analysis results

---

## Activity 5 – Interpretation and Discussion

### Description
Contextual interpretation of analytical results to support decision-making and institutional
diagnosis.

### Inputs
- Analytical results
- Institutional and planning context information

### Outputs
- Diagnostic insights
- Support material for document revision and improvement
